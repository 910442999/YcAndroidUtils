# YcAndroidUtils
安卓的工具库介绍

此库主要是收集一些开发中用到的相关工具类等

目前常用的有：复制粘贴，设备信息，编码，加密解密，文件，图片，日志，数字相关，照片相关，SP存储，系统状态栏，字符串相关，自定义toast，等

增加glide 圆角 圆形 模糊等图片的加载  数据转换  自定义圆角imageView
